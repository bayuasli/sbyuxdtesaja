**RepoManager** adalah alat bantu berbasis web yang modern dan aman untuk mengelola repositori GitHub Anda secara massal. Bersihkan proyek lama, hapus fork yang tidak terpakai, dan atur ruang kerja digital Anda dengan cepat.

## âœ¨ Fitur Utama

-   **Penghapusan Massal**: Pilih banyak repositori sekaligus dan hapus hanya dalam beberapa klik.
-   **Keamanan Maksimal**: Token GitHub Anda hanya disimpan dalam memori browser (`Session Storage`). Tidak ada data yang dikirim ke server pihak ketiga.
-   **Dashboard Informatif**: Lihat statistik repositori publik vs privat Anda dengan antarmuka yang elegan.
-   **Pencarian & Filter**: Cari repositori berdasarkan nama, deskripsi, atau bahasa pemrograman.
-   **Ekspor Data**: Unduh daftar repositori Anda dalam format CSV.
-   **Open Source**: Siap dijalankan di lingkungan lokal (`Localhost`) untuk transparansi penuh.

## ğŸš€ Cara Penggunaan

1.  Dapatkan **Personal Access Token (PAT)** dari GitHub Anda di [GitHub Settings](https://github.com/settings/tokens).
    -   *Pastikan memberikan izin `delete_repo` jika ingin menggunakan fitur penghapusan.*
2.  Buka `index.html` di browser Anda.
3.  Masukkan nama pengguna GitHub dan token Anda.
4.  Gunakan **Mode Bahaya** untuk mulai memilih repositori yang ingin dihapus.
